# 📱 PWA手机版安装详细指南

## 🎯 **PTE备考App - 手机桌面版安装**

### 📋 **PWA是什么？**

**PWA (Progressive Web App)** 是一种新型Web应用技术，可以将网站像原生App一样安装到手机桌面。

**✅ 核心优势：**
- 🚫 **无需下载** - 直接浏览器安装
- 📵 **离线使用** - 没网也能管理任务
- ⚡ **快速启动** - 桌面图标一键打开
- 🖥 **原生体验** - 全屏显示，无浏览器干扰
- 🔄 **自动更新** - 始终使用最新版本
- 💾 **数据安全** - 本地存储，隐私保护
- 📱 **跨平台** - iPhone/Android都能用

---

## 🤖 **安卓手机安装**

### 🔥 **方法一：自动安装（推荐）**
1. **浏览器打开**：Chrome、Firefox、Edge等现代浏览器
2. **访问地址**：双击 `index.html` 或在浏览器中打开
3. **等待提示**：3秒后浏览器自动弹出"添加到主屏幕"提示
4. **点击安装**：点击"安装"或"添加到主屏幕"按钮
5. **完成**：桌面出现"PTE备考"图标 ✅

### 🔧 **方法二：手动安装**
1. **打开浏览器**：Chrome浏览器
2. **访问应用**：打开 `index.html`
3. **菜单操作**：点击右上角三个点（⋮）
4. **选择添加**：点击"添加到主屏幕"或"安装应用"
5. **确认安装**：点击"添加"或"安装"按钮
6. **完成**：桌面显示PTE备考图标

---

## 🍎 **iPhone/iPad安装**

### 🔥 **方法一：Safari安装**
1. **必须用Safari**：不能用其他浏览器
2. **访问应用**：Safari浏览器打开 `index.html`
3. **分享按钮**：点击底部中间的分享按钮（📤 向上箭头）
4. **添加主屏幕**：在弹出的菜单中找到并点击"添加到主屏幕"
5. **确认信息**：确认应用名称"PTE备考"
6. **点击添加**：点击右上角"添加"按钮
7. **完成**：桌面出现PTE备考图标 ✅

### 🔧 **方法二：设置安装**
1. **打开设置**：iPhone 设置 → 通用 → VPN与设备管理
2. **找到描述文件**：下滑找到"描述文件"部分
3. **信任应用**：找到开发者应用，点击信任
4. **验证安装**：返回桌面点击图标

---

## 📲 **安装后使用方法**

### 🚀 **首次启动**
1. **点击图标**：桌面点击"PTE备考"图标
2. **全屏体验**：应用以全屏模式打开，无浏览器栏
3. **初始化任务**：首次使用会显示任务初始化提示
4. **确认添加**：点击"初始化2025最新版任务"

### 📱 **界面操作**
- **滑动切换**：左右滑动切换阶段和分类
- **点击选择**：点击任务完成复选框
- **长按菜单**：长按任务可删除或编辑
- **下拉刷新**：下拉页面刷新任务列表

## 🔧 **安装问题解决**

### ❌ **安装按钮不显示**
**安卓**：
- 确保使用Chrome/Edge等现代浏览器
- 尝试刷新页面
- 清除浏览器缓存

**苹果**：
- 必须使用Safari浏览器
- 确保iOS版本在11.3以上
- 检查是否开启了限制

### ❌ **图标不显示**
- 清理浏览器缓存
- 重新添加到主屏幕
- 检查是否被系统隐藏

### ❌ **无法离线使用**
- 首次安装需要有网络连接
- 确保完全加载所有页面内容
- 重新安装一次应用
