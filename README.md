## 🎯 **45个任务详细分布**

### 📍 **阶段1: 打磨武器 (12个任务)**
**P1死磕 (4个)**:
- RS (Repeat Sentence) - 练习50%内容复述
- WFD (Write From Dictation) - 高频300句拼写训练
- RA (Read Aloud) - 每日5-10篇流利度训练
- SGD (Summarise Group Discussion) - 小组讨论总结 (新增)
- RTS (Respond to a Situation) - 情景回答 (新增)

**P2模板 (6个)**:
- DI (Describe Image) - 自然模板，避免机械背诵
- RL (Retell Lecture) - 个人理解+逻辑归纳
- WE (Writing Essay) - 内容0-6档，注重批判性思维
- SST (Summarize Spoken Text) - 个人理解+关键词提取
- SWT (Summarize Written Text) - 逻辑归纳，避免句式模板化

**P3随缘 (2个)**:
- ASQ (Answer Short Question) - 快速过题库
- FIB-W (Fill in the Blanks Writing) - 高频词汇复习

### 📍 **阶段2: 饱和攻击 (13个任务)**
**P1死磕 (4个)**:
- RS进阶训练 - 提升到70%内容复述准确率
- WFD三遍复习 - 理解→拼写→盲听，确保90%+准确率
- WFD错词本建立 - 整理所有错词死磕拼写
- WFD试词法训练 - 结尾多写复数形式提高命中率

**P2模板 (2个)**:
- SST精练 - 每天做2篇，训练提取关键词能力
- SST关键词库 - 整理高频学术词汇和连接词

**P3随缘 (7个)**:
- FIB-R (Fill in the Blanks Reading) - 每天看20篇机经
- FIB-R逻辑训练 - 理解挖空处语法和词汇搭配逻辑
- FIB-R高频词汇 - 整理常见固定搭配和词组
- RO (Re-order Paragraphs) - 学习找"指代词对"技巧
- RO保分策略 - 考试时找1-2对指代词即可
- RA持续练习 - 每日坚持5篇首读，保持新鲜感
- FIB-L (Fill in the Blanks Listening) - 独立训练，不再为写作供分

### 📍 **阶段3: 实战演习 (15个任务)**
**P1死磕 (3个)**:
- WFD最后冲刺 - 高频300句第3遍复习，确保95%+准确率
- RA考前保温 - 每天练习保持口腔肌肉状态
- RS考前练习 - 保持耳朵敏感度和口腔反应速度

**P2模板 (7个)**:
- 全真模拟测试 - 进行官方或平台Full Mock Test
- 模拟考分析 - 针对卡壳环节做专项脱敏训练
- 体力训练 - 确保2小时高度专注，不疲劳
- 硬件检查 - 确认耳机麦克风摆放位置
- 防喷麦测试 - 练习说话时避免气流直接冲击麦克风
- 心态调优 - 模拟考成绩稳定在60-65波动即可
- 考试流程熟悉 - 了解每个题型的时间和界面操作

**P3随缘 (5个)**:
- ASQ (Answer Short Question) - 快速过一遍题库，有印象即可
- FIB-W (Fill in the Blanks Writing) - 复习高频词汇拼写
- WE模板最终检查 - 确保能在压力下流利打字完成
- DI/RL模板最终检查 - 闭眼也能0.8倍速流利背诵
- SST模板最终检查 - 关键词提取和逻辑套用熟练
