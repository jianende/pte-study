# 🤔 PTE备考App - 常见问题解答

## 📱 **安装和访问问题**

### ❌ **Q: 双击index.html没有反应**
**A**: 
1. 确保文件关联到浏览器
2. 右键选择"打开方式" → 选择浏览器
3. 或者先打开浏览器，再拖拽文件进去

### ❌ **Q: 无法访问文件**
**A**:
1. 检查文件是否完整上传
2. 确保index.html在根目录
3. 尝试不同浏览器（Chrome/Firefox/Safari）
4. 清除浏览器缓存后重试

### ❌ **Q: PWA安装提示不弹出**
**A**:
1. 等待3-5秒，有时需要加载时间
2. 检查浏览器是否支持PWA
3. 在Chrome/Edge中查看地址栏是否有安装图标
4. 手动在菜单中找"添加到主屏幕"选项

---

## 📱 **PWA功能问题**

### ❌ **Q: 安装后无法离线使用**
**A**:
1. 首次安装需要网络连接
2. 确保完全加载所有页面内容
3. 检查Service Worker是否正确注册
4. 尝试重新安装一次

### ❌ **Q: 图标显示异常**
**A**:
1. 清理浏览器缓存
2. 重新添加到主屏幕
3. 检查manifest.json中的图标配置
4. 重启手机后重新安装

### ❌ **Q: 全屏模式不生效**
**A**:
1. 确保以"standalone"模式启动
2. 检查manifest.json中的display设置
3. 在iOS中检查"添加到主屏幕"是否成功
4. 尝试重新安装应用

---

## 💾 **数据存储问题**

### ❌ **Q: 任务数据丢失**
**A**:
1. localStorage存储，清除浏览器数据会丢失
2. 建议定期截图保存进度
3. 可以导出任务列表到其他位置
4. 避免在不同浏览器间切换

### ❌ **Q: 无法保存任务**
**A**:
1. 检查浏览器是否禁用了localStorage
2. 确认JavaScript没有报错
3. 尝试添加任务后刷新页面
4. 检查存储空间是否足够

### ❌ **Q: 数据同步问题**
**A**:
1. PWA版本数据存储在本地，不支持云同步
2. 同一台设备上的同一浏览器会同步
3. 不同设备需要分别安装和管理
4. 可以手动导出导入数据

---

## 🌐 **部署问题**

### ❌ **Q: GitHub Pages 404错误**
**A**:
1. 确保index.html在仓库根目录
2. 检查分支是否为main（非master）
3. 验证Pages设置中的文件夹选择
4. 等待2-5分钟部署完成

### ❌ **Q: 部署后样式丢失**
**A**:
1. 检查styles.css文件路径
2. 确保使用相对路径引用
3. 验证CSS文件是否正确上传
4. 检查浏览器控制台错误信息

### ❌ **Q: PWA功能在线部署后失效**
**A**:
1. 确保manifest.json在根目录
2. 检查Service Worker路径配置
3. 验证HTTPS访问（PWA要求）
4. 检查域名权限设置

---

## 📱 **移动端问题**

### ❌ **Q: 界面显示异常**
**A**:
1. 更新手机系统到最新版本
2. 更新浏览器到最新版本
3. 清除浏览器缓存
4. 尝试其他浏览器

### ❌ **Q: 触摸不灵敏**
**A**:
1. 清理手机屏幕
2. 重启浏览器应用
3. 确保没有屏幕保护膜影响
4. 尝试使用外接键盘操作

### ❌ **Q: 文字显示重叠**
**A**:
1. 检查浏览器字体设置
2. 调整手机显示缩放比例
3. 重启浏览器
4. 使用移动设备调试工具检查

---

## 🔧 **技术问题**

### ❌ **Q: JavaScript报错**
**A**:
1. 打开浏览器开发者工具（F12）
2. 查看Console标签中的错误信息
3. 根据错误提示定位问题
4. 检查script.js文件完整性

### ❌ **Q: Service Worker注册失败**
**A**:
1. 确保sw.js在根目录
2. 检查Service Worker语法是否正确
3. 验证文件路径配置
4. 检查HTTPS协议（本地测试可能受限）

### ❌ **Q: 缓存问题**
**A**:
1. 强制刷新页面（Ctrl+F5）
2. 清除浏览器缓存
3. 卸载PWA后重新安装
4. 使用开发者工具清除应用缓存

---

## 🎯 **功能使用问题**

### ❌ **Q: 任务无法删除**
**A**:
1. 刷新页面后重试
2. 检查是否点击了正确的删除按钮
3. 确认JavaScript没有报错
4. 尝试长按任务查看删除选项

### ❌ **Q: 筛选功能不工作**
**A**:
1. 检查是否选择了正确的筛选条件
2. 确认任务数据包含对应分类
3. 尝试重置筛选到"全部"
4. 检查标签切换是否正常响应

### ❌ **Q: 统计数据不更新**
**A**:
1. 手动刷新页面
2. 检查任务是否正确标记完成
3. 验证本地存储是否正常工作
4. 重新初始化任务数据

---

## 🔍 **性能问题**

### ❌ **Q: 应用加载慢**
**A**:
1. 检查网络连接状态
2. 清除浏览器缓存
3. 关闭不必要的浏览器标签
4. 重启浏览器或手机

### ❌ **Q: 内存占用高**
**A**:
1. 定期清理浏览器数据
2. 关闭不使用的浏览器标签
3. 重启浏览器释放内存
4. 定期重启手机

### ❌ **Q: 电量消耗快**
**A**:
1. 启用省电模式
2. 调暗屏幕亮度
3. 关闭后台应用
4. 不使用时完全关闭PWA

---

## 🛠️ **兼容性问题**

### ❌ **Q: 老版本浏览器不支持**
**A**:
1. 推荐升级到最新版本浏览器
2. Chrome: 70+, Firefox: 65+, Safari: 11.3+
3. Edge: 79+ 或使用新版本Edge
4. 不支持的浏览器使用桌面版

### ❌ **Q: 某些功能不工作**
**A**:
1. 检查浏览器是否启用了JavaScript
2. 确保没有禁用localStorage
3. 检查是否有浏览器插件干扰
4. 尝试在隐私/无痕模式下测试

---
